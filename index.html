CSS селектори

  • За ID - Індексує вказаний файл для наступного коміту
  • За класом - Фіксує проіндексовані зміни та зберігає їх в історію версій
  • За кількома класами - Перераховує всі нові або змінені файли, які потребують фіксації
  • За вкладенністю - Переводить на іншу гілку
  • За назвою елементу - Завантажує всі зміни локальної гілки у віддалений репозиторій

Селектор - це частина CSS-правила, яка повідомляє браузеру, до якого елементу веб-сторінки буде застосований CSS стиль.
