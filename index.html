CSS селектори

  • За ID Ідентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього через скрипти.
  • За класомКласи застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки.
  • За кількома класамиДо будь-якого елемента одночасно можна додати кілька класів, перераховуючи їх в атрибуті class через пробіл. У цьому разі до елемента застосовується стиль, описаний у правилах для кожного класу.
  • За вкладенністю (Дочірні)Дочірнім називається елемент, який безпосередньо розташовується всередині батьківського елемента.
  • За назвою елементуЯк селектор може виступати будь-який елемент HTML, для якого визначаються правила форматування, такі як: колір, фон, розмір тощо.
  • За атрибутомВстановлює стиль для елемента, якщо задано специфічний атрибут. Його значення в цьому випадку не важливе.

Селектор - це частина CSS-правила, яка повідомляє браузеру, до якого елементу веб-сторінки буде застосований CSS стиль.
