CSS селектори

  • За ID - Ідентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього через скрипти.
  • За класом - Класи застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки.
  • За кількома класами - До будь-якого елемента одночасно можна додати кілька класів, перераховуючи їх в атрибуті class через пробіл. У цьому разі до елемента застосовується стиль, описаний у правилах для кожного класу.
  • За вкладенністю (Дочірні) - Дочірнім називається елемент, який безпосередньо розташовується всередині батьківського елемента.
  • За назвою елементу - Як селектор може виступати будь-який елемент HTML, для якого визначаються правила форматування, такі як: колір, фон, розмір тощо.
  • За атрибутом - Встановлює стиль для елемента, якщо задано специфічний атрибут. Його значення в цьому випадку не важливе.

У цьому прикладі змінюється колір тексту і вирівнювання тексту абзацу. Стиль застосовуватиметься тільки до тексту, який розташовується всередині контейнера

Під час опису ідентифікатора спочатку вказується символ решітки (#), потім іде ім'я ідентифікатора.
